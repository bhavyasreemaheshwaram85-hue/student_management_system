# Student Management System (Java GUI)

## Project Description

The **Student Management System** is a simple Java GUI application developed using **Swing and AWT**.
This application allows users to manage student records by performing operations such as **Add, View, Search, Update, and Delete**.

The program stores student data using an **ArrayList** and provides an interactive graphical interface for easy access and management.

---

## Features

* Add new student details
* View all student records
* Search student by ID
* Update student information
* Delete student record

---

## Technologies Used

* Java
* Swing (GUI components)
* AWT (layout management)
* ArrayList (data storage)

---

## Student Information Stored

Each student record contains:

* Student Name
* Student ID
* Age
* Department
* Semester
* Contact Number

---

## How the Application Works

1. The program opens a **GUI window** with buttons for different operations.
2. Users can click the buttons to perform actions.
3. Student data is stored temporarily in an **ArrayList**.
4. The system displays results in a **text area**.

---

## Functionalities

### Add Student

Allows the user to enter student details through a dialog box and store them in the system.

### View Students

Displays all stored student records in the display area.

### Search Student

Allows the user to search for a student using the **Student ID**.

### Update Student

Updates existing student details by entering the student ID.

### Delete Student

Deletes a student record from the system using the student ID.

---

## How to Run the Program

1. Open the project in a Java IDE such as Eclipse.
2. Copy the source code into a file named:

```
StudentManagementGUI.java
```

3. Compile the program:

```
javac StudentManagementGUI.java
```

4. Run the program:

```
java StudentManagementGUI
```

The program opens a graphical interface with buttons:
